Что такое JavaScript?

JavaScript — это язык сценариев, используемый для создания и управления динамическим контентом веб-сайта, т.е. всем, что перемещается, обновляется или иным образом изменяется на вашем экране, не требуя перезагрузки веб-страницы вручную.
Для JS создаются различные фреймворки и библиотеки, предназначенные для упрощения использования данного языка программирования.

Библиотеки JavaScript представляют собой наборы заранее написанных фрагментов кода, которые можно использовать для выполнения общих функций JavaScript.
Фреймворки JavaScript представляют собой полный набор инструментов, которые помогают формировать и организовывать ваш веб-сайт или веб-приложение.

Область использования этого языка очень обширна, его используют для достижения таких целей как:

  1. добавление интерактивности на веб-сайт
  2. разработка мобильных приложений
  3. создание игр на основе веб-браузера
  4. back-end разработка

К преимуществам данного языка можно отнести скорость, популярность, универсальность, JS повышает производительность веб-сайтов и приложений за счет уменьшения длины кода.
К недостаткам поддержка браузера – каждый браузер по-разному интерпретирует JavaScript, также не каждый браузер может поддерживать новые функции обновленной версии языка.

Классический JavaScript — это язык программирования для интернета, он бессилен за пределами браузера. С помощью JavaScript нельзя запустить программу на компьютере или записать файл в нужную папку.
